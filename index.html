JavaScript Chapters 9-25 Summary

Chapter 9-11: User Input & Conditional Statement

User Input

User input in JavaScript can be taken using the prompt() function or through form elements in HTML.

const name = prompt("Enter your name:"); alert(`Hello, ${name}!`);

Conditional Statements

Conditional statements control the flow of the program based on conditions:

  • if statement: Executes a block of code if a specified condition is true.
  • else statement: Executes a block of code if the condition in the if statement is false.
  • else if statement: Tests multiple conditions, executing the corresponding block of code for the first true condition.
const age = prompt("Enter your age:");
if (age < 18) {
  alert("You are a minor.");
} else if (age < 65) {
  alert("You are an adult.");
} else {
  alert("You are a senior.");
}

Chapter 12-13: IF...ELSE & ELSE IF Statement, Testing Set of Conditions

IF...ELSE & ELSE IF Statement

The if...else and else if statements are used to execute code based on specific conditions.

const score = 85;
if (score >= 90) {
  console.log("Grade: A");
} else if (score >= 80) {
  console.log("Grade: B");
} else if (score >= 70) {
  console.log("Grade: C");
} else {
  console.log("Grade: F");
}

Testing Set of Conditions

Testing a set of conditions can be done using multiple else if statements or using logical operators within a single if statement.

const day = "Monday";
if (day === "Saturday" || day === "Sunday") {
  console.log("It's a weekend!");
} else {
  console.log("It's a weekday.");
}

Chapter 14-16: Arrays and Loop

Arrays

Arrays are used to store multiple values in a single variable:

  • Array literals: Define arrays using square brackets.
  • Array methods: Perform operations on arrays (e.g., push, pop, shift, unshift).
const fruits = ["Apple", "Banana", "Cherry"];
fruits.push("Orange");
console.log(fruits); // ["Apple", "Banana", "Cherry", "Orange"]

Loop

Loops execute a block of code multiple times:

  • for loop: Repeats a block of code a certain number of times.
  • while loop: Repeats a block of code as long as a specified condition is true.
  • do...while loop: Executes a block of code once before checking if the condition is true, then repeats the loop as long as the condition is true.
for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}

Chapter 20-25: String Methods

String Methods

String methods are used to perform operations on strings:

  • length: Returns the length of a string.
  • toUpperCase(): Converts a string to uppercase letters.
  • toLowerCase(): Converts a string to lowercase letters.
  • slice(start, end): Extracts a part of a string and returns it as a new string.
  • substring(start, end): Extracts characters from a string, between two specified indices.
  • replace(searchValue, newValue): Replaces occurrences of a substring with a new substring.
  • trim(): Removes whitespace from both ends of a string.
const text = " Hello, World! ";
console.log(text.length); // 17
console.log(text.toUpperCase()); // " HELLO, WORLD! "
console.log(text.toLowerCase()); // " hello, world! "
console.log(text.slice(2, 7)); // "Hello"
console.log(text.replace("World", "JavaScript")); // " Hello, JavaScript! "
console.log(text.trim()); // "Hello, World!"
